description: "How to use TeamWork to automate your Chrome Browser"
---

Computer-Use in TeamWork currently works as a browser automation in Google Chrome. It does not yet mean full desktop control of Ubuntu, macOS, or Windows apps.

With the right skills, your TeamWork agent will be able to runs browser actions and take screnshots as proof of having completed said actions. If you connect to a remote TeamWork server, that remote machine also needs to have Chrome installed.

Today, the supported path is `Control Chrome`, which uses Chrome DevTools MCP.

## Requirements

- Google Chrome on the machine running the session
- In `Settings > Extensions > Control Chrome`
